# Prework Note Keeping Website

## Description

This project was built with the intention of keeping notes from the bootcamp organized and easily accessable for future reference.

This project also provided a place to explore initial concepts of HTTP, CSS, JavaScript, and Git.

## Usage

the website can be used by navigating to 127.0.0.1:5500/index.html. Notes are organized into sections and can be added to by editing the HTML document in the PreworkStudyGuide folder.
